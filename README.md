# slovak-cities-coordinates
A complete list of the coordinates of Slovak towns and cities in various formats. Coordinates system (CRS): EPSG:4326, WGS 84.

#### Available formats

- GeoJSON
- JSON
- SHP
- SQL
- GML
- GPX
- KML
- TXT
- XLS
